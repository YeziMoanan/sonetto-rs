# CN 3.8 Handoff And Continuation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 固化国服 3.8 仿真服务端的需求、已完成实现、验证证据和后续开发顺序，确保后续工作从当前证据继续，不重复分析，也不把尚未完成的客户端闭环误报为完成。

**Architecture:** `sonetto-rs-3.8-cn` 是从可运行的国际服 3.6 服务端复制出的独立项目，使用独立端口、运行目录和数据库。国服 Android 客户端仍需通过 `sonetto-patch` 中既有的 Android 3.8 trace/redirect 方案，将经过人工确认的 HTTP 和游戏 TCP 端点精确转发到本地；不使用系统代理、DNS、hosts 或证书改动。

**Tech Stack:** Rust 2024、Axum 0.8、Tokio、SQLx/SQLite、MuMu Player 12、MuMuManager/ADB、既有 `sonetto-patch` Android Frida 方案。
