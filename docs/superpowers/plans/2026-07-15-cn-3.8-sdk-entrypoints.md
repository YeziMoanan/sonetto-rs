# CN 3.8 SDK Entrypoints Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 让国服 `gameId=50001` 客户端命中现有 SDK 处理器，并让 SDK 初始化返回国服渠道 `100`，同时保留国际服 `60001/200` 兼容入口。

**Architecture:** 路由层为已验证的 game-id 路径注册显式别名，不复制处理器。SDK 初始化从查询参数读取 `gameId` 和可选 `channelId`，只在缺失时采用项目默认值；空查询的登录菜单响应保持不变。

**Tech Stack:** Rust 2024、Axum 0.8、Tower ServiceExt、Serde JSON、现有 sdkserver 单元测试。
